Tools for blogging - less is more

March 13, 2021

Blogging is old-school now, with Youtube channels, Instagram stories, podcasts, live streaming (Twitch, Clubhouse) and newsletters (hello Substack!). But I like retro things…

For my blog revival, I reviewed a few different tools. A lot of solutions are available, especially iecause I like DIY and because I am a developper, I have a lot of solutions available.

Today, the total cost to get my blog up and running is free. I would just need to pay for a domain name and I found one super cheap at $3 per month because it has a strange extension: .co.

To create a blog you need to choose 3 things:

  • a domain name (namecheap.com, godaddy.com)
  • a host: where you store your website (Github pages, AWS, Notion, Ghost)
  • a technology: how you build your pages (Hugo, Jekyll, no-code)

Why use a domain name?

  • To get all the hard-won traffic and SEO associated with your own domain name.
  • To not confuse your audience with different domains.
  • No hidden advertisement.

No code with Notion

Without any hesitations I would recommend to people with no code experience to think about using Notion for their blogging platform. Notion offers a no code solution to publish webpages. For $5 per site per month, you can be up and running and you can use your own domain name. I think that this is a great way to start so you can focus on creating quality content regularly.

A minimalist personal website: https://mariohayashi.com
Another one: https://jodieloi.com

Total monthly cost of Notion: $4 (best case) for the domain name + $5 for Notion = $9 per month

No code with Ghost

Ghost is THE alternative to Wordpress: everything is better! It is lightning fast and safe. The integrated interface is great to manage content (CMS).

They are offering new ways to monetize content for serious bloggers: articles behind a paywall, paid newsletters.

I like that Ghost is a non-profit and that they are trying to change the way we create and consume content. I like the genuine relationship that can be build between the content creator and the reader. The publisher can focus on creating great content, there is no need to work on trying to monetize content by looking for affiliation, etc. The reader can focus on the content. He does not need to read between lines and be careful about hidden affiliations.

Total monthly cost of Ghost (no code): $4 for the domain name + $29 per month but there is no percentage taken on the paid subscriptions = $33 per month

For developers, there is an open-source version of Ghost. You still need to find a CDN to host your static website. DigitalOcean has a pre-built ghost image set up that is working well for $5 per month. AWS Lightsail is another solution for $3.5 per month.

Total monthly cost of Ghost (developer edition): $4 for the domain name + around $4 for hosting = $8 per month

Hugo (for developers)

Hugo is a static website generator. This is the tool that I am using for this blog. I like that it is light and minimalist. I host my website on Github pages. It is free as well but there are some traffic limitations, so if my blog is too successful I would need to migrate my blog to AWS for example.

Total monthly cost of Hugo (developer edition): $4 per month for the domain name

Here is the ressource I used to set up my website: https://www.mytechramblings.com/posts/create-a-website-with-hugo-and-gh/

I use the following commands to publish my posts and then it is live!

hugo --theme lithium
cd public
git add *
git commit -m 'Add a new post'
git push -u origin master

Jekyll (for developers)

Jekyll is very similar to Hugo, it is a static website generator and it is free. It is very well integrated with Github pages, so updating your website once everything is set up is a breeze!

Total monthly cost of Jekyll (developer edition): $4 per month for the domain name
